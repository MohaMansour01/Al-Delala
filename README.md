🛒 Al-Delala Marketplace

Al-Delala is a modern digital marketplace designed to connect buyers and sellers in a fast, simple, and reliable way. The platform allows users to browse listings, post products, and interact seamlessly in a user-friendly environment.

Built with performance and scalability in mind, Al-Delala aims to provide a smooth experience across devices while keeping the interface clean and intuitive.

⚙️ Frontend Tech Stack

This project is built using modern web technologies:

Next.js – React framework for building fast and scalable web apps
TypeScript – Strongly typed JavaScript for better code quality
Tailwind CSS – Utility-first CSS for rapid UI development
App Router – Modern routing system in Next.js
ESLint – Code quality and consistency
